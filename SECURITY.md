# Security Policy

# Politique de sécurité

## Contexte

Ce dépôt est un projet académique d’analyse du réseau Tor et des services .onion.

Aucune application en production n’est déployée via ce dépôt.

## Signalement d’une vulnérabilité

Si vous identifiez une vulnérabilité ou un problème de sécurité concernant ce dépôt :

- Merci d’ouvrir une issue GitHub
- Ou de contacter l’auteur via GitHub

## Remarque

Ce projet est réalisé à des fins pédagogiques uniquement.
Aucune activité illégale n’est encouragée ou réalisée.
